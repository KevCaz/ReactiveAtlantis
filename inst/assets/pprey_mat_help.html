Summary

This program displays data for the predator prey relationship for Atlantis run. Also, provide help for the tuning of the pprey matrix

Details

Plots have a zoom feature. Draw a box and double click to zoom into the box. Double click to reset zoom.

Using the input panel, the user can select the focal prey (Prey panel) and predator Predator panel .

The Original value : box, display the value that is used on the pprey matrix on the Biological parameter file.

The Current value : box, display the value that the user set in the current run. If the user don't save the run, the value will be lost.

On the New value : box, the user can enter the new value for the pprey matrix for the selected predator and prey. The result of the application would be reflected on the current run in all the plots after the used click the box Change value.

The Write pPREY Matrix bottom create a txt file that contain the new pPrey matrix created by the user

Effective predation Represent the total predation based on the Predator's biomass, is the same approach that Beth used on the spreadsheet to calibrate predation. Values are on logarithmic scale

Availability matrix The raw pPREY matrix from the biological parameter file.

Overlap matrix shows if the predator is able to eat the prey based on the gape limitations.

% of predation pressure Which percentage of each prey corresponds to the total consumed by the predator.

Total biomass prey Total biomass of each functional group on logarithmic scale.
